2. 人工智能的**主要学派**:

   1. **符号主义**：认为人工智能源于数理逻辑。该学派将数学严格公理化，从公理出发，由逻辑推理得到引理，定理，推论。人工智能的创始人之一，John McCarthy 是符号主义学派的拥护者

   3. **行为主义**：来源于控制论及“感知—动作”型控制系统。该学派认为智能取决于感知和行动，人工智能可以像人类智能一样逐步进化，以及智能行为只能在现实世界中与周围环境交互作用而表现出来。强调系统与环境的交互，从运行环境中获取信息，通过自己的动作对环境施加影响

## 逻辑推理基础

### 1.1 逻辑推理的核心定义

- 逻辑推理：指用**蕴涵**推导出结论的过程；与之对应的`模型验证`，是通过枚举所有可能的模型，验证在给定知识库KB的前提下，结论始终为真。
- 形式化表示：若推理算法可以根据KB导出结论α，记为 KB⊢α。
- 可靠推理：只会导出蕴涵句的推理算法，被称为**可靠的 / 真值保持的**。

### 1.2 三大核心逻辑推理方法

| 推理类型                        | 核心逻辑                                      | 推理方向  | 核心示例（豆子案例）                                         |
| :------------------------------ | :-------------------------------------------- | :-------- | :----------------------------------------------------------- |
| 演绎推理（Deductive Reasoning） | 从一般规则推导出个别结论，三段论式核心        | 一般→个别 | 规则：袋子中豆子都是白色的；情况：豆子来自这个袋子；结果：这些豆子是白色的 |
| 归纳推理（Inductive Reasoning） | 从个别现象总结出一般规则，分完全 / 不完全归纳 | 个别→一般 | 情况：豆子来自这个袋子；结果：这些豆子是白色的；规则：袋子中豆子都是白色的 |
| 溯因推理（Abductive Reasoning） | 从结果反推导致结果的最可能情况                | 结果→原因 | 规则：袋子中豆子都是白色的；结果：这些豆子是白色的；情况：豆子来自这个袋子 |

#### 演绎推理经典三段论示例

- 大前提：足球运动员的身体都是强壮的
- 小前提：高波是一名足球运动员
- 结论：高波的身体是强壮的

### 1.3 自然演绎推理

#### 核心定义

从一组已知为真的事实出发，运用经典逻辑的推理规则推出结论的过程。

#### 核心推理规则

1. 假言推理：

   P,P→Q/Q

   - 示例：“如果 x 是金属，则 x 能导电”，“铜是金属”，推出 “铜能导电”

2. 拒取式推理：

   P→Q,¬Q/¬P

   - 示例：“如果下雨，则地下就湿”，“地上不湿”，推出 “没有下雨”

   

3. P 规则：在推理的任何步骤都可以引入前提

4. T 规则：在推理中，若前面有一个或多个公式永真蕴涵公式 S，则可把 S 引入推理过程

#### 推理实例

**已知事实**：

1. 凡是容易的课程小王 (wang) 都喜欢

2. C 班的课程都是容易的

3. ds 是 C 班的一门课程

   

   求证

   ：小王喜欢 ds 这门课程

**证明过程**：

1. 定义谓词

   - Easy(x)：x 是容易的
   - Like(x,y)：x 喜欢 y
   - C(x)：x 是 C 班的一门课程

   

2. 谓词公式表示

   - 前提 1：(∀x)(Easy(x)→Like(wang,x)
   - 前提 2：(∀x)(C(x)→Easy(x)
   - 前提 3：C(ds)
   - 待证结论：Like(wang,ds)

   

3. 应用推理规则

   1. 全称实例化：(∀x)(Easy(x)→Like(wang,x)⟹Easy(z)→Like(wang,z)
   2. 全称实例化：(∀x)(C(x)→Easy(x)⟹C(y)→Easy(y)
   3. P 规则 + 假言推理：C(ds),C(y)→Easy(y)⟹Easy(ds)
   4. T 规则 + 假言推理：Easy(ds),Easy(z)→Like(wang,z)⟹Like(wang,ds)

   

#### 自然演绎推理的优缺点

- **优点**：定理证明过程自然，易理解；推理规则丰富，过程灵活；便于嵌入领域启发式知识
- **缺点**：易产生组合爆炸，得到的中间结论一般呈指数形式递增

## 二、归结演绎推理基础

### 2.1 核心原理：反证法

- 核心等价式：P→Q 当且仅当 P∧¬Q↔False
- 定理：Q 为 P1,P2,…,Pn 的逻辑推论，当且仅当 (P1∧P2∧...∧Pn)∧¬Q 是不可满足的
- 核心思路：证明 P→Q 等价于证明子句集 S={P1,P2,...,Pn,¬Q} 是不可满足的

### 2.2 核心概念

1. **文字**：原子公式及其否定；P 为正文字，¬P 为负文字

2. **子句**：任何文字的析取式，单个文字本身也是子句；例：P∨¬Q 记作 (P,¬Q)

3. **空子句**：不包含任何文字的子句，记作 NIL；空子句永假、不可满足

4. **子句集**：由子句构成的集合（子句间为合取关系）；例：(P∨¬Q)∧(R∨S) 记作 {(P,¬Q),(R,S)}

5. 归结式：

   对于任意两个子句 C1 和 C2，若 C1 中有文字 L，C2 中有互补文字 ¬L，则删去 L 和 ¬L，将剩余部分组成新的析取式，即为归结式。

   - 示例：子句 (W,R,Q) 和 (W,S,¬R) 的归结式为 (W,Q,S)
   - 核心定理：子句 P 和 ¬P 的归结式为空子句

   

### 2.3 鲁宾逊归结原理

- 子句集中子句之间是合取关系，只要有一个子句不可满足，子句集就不可满足

- 判定规则：

  1. 检查子句集S中是否包含空子句，若包含，则S不可满足
  2. 若不包含，在S中选择合适的子句进行归结，一旦归结出空子句，说明S不可满足

## 三、命题逻辑的归结推理

### 3.1 归结推理步骤

1. 把前提集F转化成子句集表示，得到子句集S~0~
2. 把待证命题P的否定式¬P转化成子句集表示，并入S~0~，得到S=S~0~∪S~¬P~
3. 对子句集S反复应用归结推理规则，直至导出空子句，证明结束（出现空子句即证明原命题成立）

### 3.2 推理实例

**已知前提**：

1. P

2. (P∧Q)→R

3. (S∨T)→Q

4. T

   求证：R

**证明过程**：

1. 转化为子句集：S={P,¬P∨¬Q∨R,¬S∨Q,¬T∨Q,T,¬R}
2. 归结过程：
   1. ¬P∨¬Q∨R 与 ¬R 归结，得 ¬P∨¬Q
   2. ¬P∨¬Q 与 P 归结，得 ¬Q
   3. ¬Q 与 ¬T∨Q 归结，得 ¬T
   4. ¬T 与 T 归结，得**空子句 NIL**
3. 结论：出现空子句，命题R得证

## 四、谓词逻辑的归结推理

### 4.1 置换与合一

#### 1. 置换

- 定义：用项替换变量的操作，形式化表示为 σ={x1/t1,x2/t2,...,xn/tn}，表示用ti替换变量xi
- 置换合成：对表达式依次用θ和σ置换，等价于先合成置换θσ，再用合成置换对表达式置换，即 (Sθ)σ=S(θσ)

#### 2. 合一与最一般合一项（MGU）

- 合一：寻找项之间合适的变量置换，使两个表达式一致的过程；若置换σ使fσ=gσ，则σ是f和g的合一项
- 最一般合一项（MGU）：若σ是f和g的合一项，且对任意其他合一项θ，都存在替换λ使得θ=σλ，则σ为最一般合一项

#### 3. 最一般合一项求解算法

1. 初始化：σ=∅，S={f,g}

2. 若S包含相同的语句，算法停止，σ即为 MGU

3. 否则，找出S的差异集D={e1,e2}：

   - 若e1是变量v，e2是不包含v的项t，则令σ=σ{v/t}，S=S{v/t}，返回步骤 2
   - 否则，算法停止，f和g不可合一

   

#### 4. 求解实例

**示例 1**：求P(f(x),z)和P(y,a)的 MGU

1. 初始化：σ=∅，S={P(f(x),z),P(y,a)}
2. 差异集D={f(x),y}，更新σ={y/f(x)}，S={P(f(x),z),P(f(x),a)}
3. 差异集D={z,a}，更新σ={y/f(x),z/a}，S={P(f(x),a),P(f(x),a)}
4. 最终 MGU：σ={y/f(x),z/a}

**示例 2**：求P(a,x,h(g(z))和P(z,h(y),h(y)的 MGU

1. 引入合一项{z/a}，得到S={P(a,x,h(g(a)),P(a,h(y),h(y)}
2. 合一项更新为{z/a,x/h(y)}，得到S={P(a,h(y),h(g(a)),P(a,h(y),h(y)}
3. 合一项更新为{z/a,x/h(g(a),y/g(a)}，得到S={P(a,h(g(a),h(g(a)),P(a,h(g(a),h(g(a))}
4. 最终 MGU：σ={z/a,x/h(g(a),y/g(a)}

### 4.2 谓词公式化为子句集的 9 个步骤

1. **消去蕴涵和等价符号**：利用P→Q⇔¬P∨Q、P↔Q⇔(P∧Q)∨(¬P∧¬Q)替换

2. **内移否定符号 ¬**：利用双重否定律、德摩根律、量词转换律，将否定符号移到紧靠谓词的位置

3. **变量标准化**：对变量换名，使每一量词只约束一个唯一的变量名

4. 消去存在量词（Skolemize）

   ：

   - 若存在量词不在任何全称量词辖域内，用 Skolem 常量替代
   - 若受全称量词约束，用 Skolem 函数替代存在量词约束的变量

   

5. **化为前束型**：将公式化为「全称量词串 + 不含量词的母式」的形式

6. **把母式化成合取范式**：反复使用结合律和分配律，将母式转化为合取范式

7. **略去全称量词**：母式的变量均受全称量词约束，可直接省略全称量词

8. **把母式用子句集表示**：将每个合取元作为一个子句，用集合形式表示

9. **子句变量标准化**：对变量重新命名，使任意两个子句无相同变量出现

### 4.3 谓词逻辑归结推理实例

**已知前提**：

1. 会朗读的人是识字的

2. 海豚都不识字

3. 有些海豚是很机灵的

   

   求证

   ：有些很机灵的东西不会朗读

**证明过程**：

1. 谓词定义与公式表示

   - R(x)：x 会朗读；W(x)：x 识字；D(x)：x 是海豚；S(x)：x 机灵
   - 前提 1：∀x(R(x)→W(x)
   - 前提 2：∀x(D(x)→¬W(x)
   - 前提 3：∃x(D(x)∧S(x)
   - 结论：∃x(S(x)∧¬R(x)

   

2. 前提与结论取反后化为子句集

   1. ¬R(x)∨W(x)
   2. ¬D(y)∨¬W(y)
   3. D(a)
   4. S(a)
   5. ¬S(z)∨R(z)（结论取反后的子句）

   

3. 归结过程

   1. 子句 4 与 5，置换{z/a}，归结得R(a)
   2. 子句 1 与R(a)，置换{x/a}，归结得W(a)
   3. 子句 2 与W(a)，置换{y/a}，归结得¬D(a)
   4. 子句 3 与¬D(a)，归结得**空子句 NIL**

   

4. 结论：出现空子句，原命题得证

### 4.4 可判定与不可判定

- **可判定问题**：存在一个算法 / 过程，可在有限步内停止并给出正确解答的问题

- **不可判定问题**：不存在上述算法 / 过程的问题；例：不存在通用过程能判定一个子句集是否是可满足的

- 谓词逻辑归结特性：

  1. 若子句集不可满足，则必存在一个从该子句集到空子句的归结推导
  2. 若从子句集存在到空子句的推导，则该子句集不可满足
  3. 若未归结出空子句，既不能判定子句集不可满足，也不能判定其可满足

## 五、归结原理的问题求解

### 5.1 问题求解步骤

1. 已知前提F用谓词公式表示，并化为子句集S
2. 待求解问题P用谓词公式表示，否定P后与answer构成析取式¬P∨answer
3. 将¬P∨answer化为子句集，并入子句集S，得到新子句集S′
4. 对S′应用归结原理进行归结
5. 若得到归结式answer，则答案就在answer中

### 5.2 求解实例

**问题**：A,B,C 三人中有人从不说真话，也有人从不说假话。某人问三人：谁是说谎者？

- A 答：B 和 C 都是说谎者

- B 答：A 和 C 都是说谎者

- C 答：A 和 B 中至少有一个是说谎者

  求

  ：谁是老实人，谁是说谎者？

**求解过程**：

1. 谓词定义：T(x)表示 x 说真话

2. 转化为谓词公式并化为子句集

   S

   ：

   1. ¬T(A)∨¬T(B)
   2. ¬T(A)∨¬T(C)
   3. T(C)∨T(A)∨T(B)
   4. ¬T(B)∨¬T(C)
   5. ¬T(C)∨¬T(A)∨¬T(B)
   6. T(A)∨T(C)
   7. T(B)∨T(C)

   

3. 求解老实人：加入子句

   ¬T(x)∨Answer(x)

   ，进行归结

   1. 子句 1 和 7 归结，得¬T(A)∨T(C)
   2. 上述结果与子句 6 归结，得T(C)
   3. T(C)与¬T(x)∨Answer(x)置换{x/C}归结，得Answer(C)

   

4. 证明 A、B 不是老实人：加入子句T(A)，归结后得到空子句，证明¬T(A)；同理可证¬T(B)

5. 最终结论：**C 是老实人，A 和 B 是说谎者**
