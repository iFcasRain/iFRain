### JavaScript是什么

①是一种运行在客户端（*浏览器*）的**编程语言**，实现人机交互效果。

②可以实现：1.*网页特效*（监听用户的一些行为让网页做出对应反馈）

​						2.*表单验证*（对表单数据合法性判断）

​						3.*数据交互*（获取后端数据，渲染到前端）

​						4.*服务端编程*（node.js)
