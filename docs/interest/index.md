# 杂学

课程之外、出于兴趣的阅读与思考记录。不追求体系完整，只记下某段时间里真正打动过自己的概念、问题和片段。

与 **笔记**（课内、技术向）和 **随笔**（更随意的生活文字）分开存放。

## 方向

| 领域 | 说明 |
|------|------|
| [哲学](./philosophy/introduction) | 形而上、伦理、认识论等零散阅读 |
| [心理学](./psychology/introduction) | 认知、发展、社会心理等科普与原著摘记 |

## 写作约定（给自己）

- 每篇开头用一两句话写**为什么此刻关心这个**
- 区分「书上的观点」和「自己的理解」，避免混成定论
- 允许未完成：问号比假装懂更有价值
