# 信号与系统

### 课程核心

- 两大模块：**信号分析**、**系统分析**
- 研究对象：**确知信号**与**线性时不变（LTI）系统**
- 核心思想：**信号分解**（将复杂信号分解为简单信号的线性组合）
- 分析方法：时域分析、变换域分析（傅里叶变换、拉普拉斯变换、z 变换）
- 先修课程：高等代数、数学分析、程序设计

### 学习意义

- 专业基础课，支撑数字信号处理、通信原理、图像处理、信息安全、计算机视觉、智能机器人等后续课程
- 应用覆盖：音视频处理、通信、控制、微电子、机械、生物、化学、地质、经济等多个领域

## 一、信号与系统基本概念

### 1. 信号

- 定义：传递消息 / 信息的**物理形式**，是承载信息的函数，记为 `x(t)`（连续时间）或 `x[n]`（离散时间）
- 常见形式：光（烽火、火炬）、声（击鼓、鸣金）、电（电报、电话、无线电）、文字/视觉（email、导航信号）、生物电（器官功能信号）、经济统计数据（社会经济发展指标）
- 自变量：时间、空间等；因变量：声压（或电压、电流）、亮度、浓度等物理量

### 2. 系统

- 定义：处理输入信号、生成输出信号的结构，可表示为输入与输出的映射关系：`y(t) = T{x(t)}`（连续）或 `y[n] = T{x[n]}`（离散），其中 `x` 为输入，`y` 为输出，`T` 为系统运算算子

### 3. 频域视角

- 信号可通过**时域 / 空域**与**频域**双重表示，在傅里叶变换意义下二者等价，即：若时域信号为 `x(t)`，其傅里叶变换（频域表示）为 `X(jω) = ∫₋∞^∞ x(t)e^(-jωt)dt`，逆变换为 `x(t) = (1/(2π)∫₋∞^∞ X(jω)e^(jωt)dω`
- 典型应用：低通滤波（去高频，保留低频信号）、高通滤波（去低频，保留高频信号）、带阻滤波（去除特定频率范围信号，用于图像复原等场景）

## 二、1.1 连续时间与离散时间信号

### 1. 按自变量分类

- **连续时间信号**：自变量（通常为时间`t`）可在某一区间内连续取值，定义域为连续区间，例如 `x(t) = sin(t)`
- **离散时间信号**：自变量（通常记为 `n`）仅取整数，定义域为整数集，记为 `x[n]`
  - 天然离散：年利率、人口统计数据（仅在整数年份取值）
  - 采样离散：CD 音频（44kHz 采样，即每秒对连续音频信号采样 44000 个点，得到离散序列 `x[n]`）

### 2. 按确定性分类

- **确知信号**：可由确定的数学函数描述，其未来取值可准确预测（课程核心研究对象），例如 `x(t) = A cos(ωt + φ)`
- **随机信号**：无确定变化规律，无法用固定函数描述，仅能通过统计特性分析（本课程不研究）

### 3. 能量与功率属性

- 能量信号：信号总能量有限（`E = ∫₋∞^∞ |x(t)|²dt < ∞`，连续时间）或 `E = Σₙ=-∞^∞ |x[n]|² < ∞`（离散时间），平均功率为 0
- 功率信号：信号总能量无限，但平均功率有限（`P = lim(T→∞) (1/(2T)∫₋T^T |x(t)|²dt < ∞`，连续时间）或 `P = lim(N→∞) (1/(2N+1)Σₙ=-N^N |x[n]|² < ∞`（离散时间），通常为周期信号
- 既非能量也非功率信号：总能量和平均功率均无限，例如 `x(t) = t`（连续时间）

### 4. 周期属性

- 连续时间周期信号：存在最小正周期 `T₀`，满足 `x(t + T₀) = x(t)` 对所有 `t` 成立，例如正弦信号 `x(t) = A cos(ω₀t + φ)`，周期 `T₀ = 2π/ω₀`
- 离散时间周期信号：存在最小正整数周期 `N₀`，满足`x[n + N₀] = x[n]` 对所有整数 `n` 成立，例如`x[n] = A cos(Ω₀n + φ)`，仅当 `Ω₀/(2π)` 为有理数时是周期信号

## 三、1.2 自变量变换

### 1. 三类基本变换（以连续时间信号 `x(t)` 为例，离散时间类似）

1. **时移**：信号沿时间轴平移，若平移量为 `t₀`，则变换后信号为 `x(t - t₀)`（`t₀ > 0` 向右平移，`t₀ < 0` 向左平移）
2. **反转**：信号关于纵轴翻转，变换后信号为 `x(-t)`
3. **尺度**：信号沿时间轴拉伸或压缩，若尺度因子为 `a`（`a > 0`），则变换后信号为 `x(at)`（`a > 1` 压缩，`0 < a < 1` 拉伸）；离散时间自变量仅取整数，**通常不做尺度变换**

### 2. 奇偶信号

- 偶信号：满足 `x(-t) = x(t)`（连续时间）或 `x[-n] = x[n]`（离散时间），图像关于纵轴对称
- 奇信号：满足 `x(-t) = -x(t)`（连续时间）或`x[-n] = -x[n]`（离散时间），图像关于原点对称
- 任意信号可分解为**偶分量 + 奇分量**，分解公式为：         连续时间：偶分量`x_e(t) = [x(t) + x(-t)]/2`，奇分量 `x_o(t) = [x(t) - x(-t)]/2` 离散时间：偶分量 `x_e[n] = [x[n] + x[-n]]/2`，奇分量 `x_o[n] = [x[n] - x[-n]]/2`

## 四、1.3 复指数信号与正弦信号

### 1. 连续时间信号

- 基础：复数极坐标表示（`z = re^(jθ) = r(cosθ + j sinθ)`）、欧拉公式（`e^(jθ) = cosθ + j sinθ`，逆公式 `cosθ = [e^(jθ) + e^(-jθ)]/2`，`sinθ = [e^(jθ) - e^(-jθ)]/(2j)`）

- 周期复指数 / 正弦信号：         

  复指数信号：`x(t) = e^(jω₀t)`，基波周期 `T₀ = 2π/ω₀` ,可通过欧拉公式转换成![](/assets/image-20260421113055799.png)

  当ω0=0 时,x(t)=C,通常称为直流信号

  正弦信号：`x(t) = A cos(ω₀t + φ)` 或 `x(t) = A sin(ω₀t + φ)`，可通过欧拉公式表示为复指数信号的线性组合：`A cos(ω₀t + φ) = [A e^(jφ) e^(jω₀t) + A e^(-jφ) e^(-jω₀t)]/2` 核心特性：复指数信号的实部、虚部均为正弦信号

- 一般复指数信号：`x(t) = A e^(st)`，其中 `s = σ + jω`（`σ` 为实部，`ω` 为虚部），该信号可看成是振幅按实指数信号 `A e^(σt)` 规律变化的周期性复指数信号，其虚部与实部分别为 `A e^(σt) cos(ωt)` 和 `A e^(σt) sin(ωt)`（振幅随 `σ` 变化：`σ > 0` 增幅，`σ < 0` 衰减，`σ = 0` 等幅）

### 2. 离散时间信号

- 正弦信号：`x[n] = A cos(Ω₀n + φ)` 或 `x[n] = A sin(Ω₀n + φ)`，其中 `Ω₀` 为离散角频率（单位：rad）
- 一般复指数信号：`x[n] = A r^n e^(jΩ₀n) = A (r e^(jΩ₀)^n`，其中 `r > 0` 为幅度因子，`Ω₀` 为离散角频率；其虚部与实部分别为`A r^n cos(Ω₀n)` 和 `A r^n sin(Ω₀n)`，幅度按 `r^n` 规律变化（`r > 1` 增幅，`r < 1` 衰减，`r = 1` 等幅）

### 3. 核心价值

几乎所有实用信号可表示为**复指数信号的叠加**（即傅里叶级数展开）：连续周期信号 `x(t) = Σₖ=-∞^∞ cₖ e^(j k ω₀ t)`（`cₖ` 为傅里叶系数），离散周期信号 `x[n] = Σₖ=0^(N₀-1) cₖ e^(j k (2π/N₀) n)`；该特性是 LTI 系统分析的核心基础，可简化系统响应的计算。

## 五、1.4 单位冲激与单位阶跃函数

### 1. 单位阶跃函数

- 连续时间单位阶跃函数：`u(t) = { 0, t < 0; 1, t ≥ 0 }`

- 离散时间单位阶跃序列：`u[n] = { 0, n < 0; 1, n ≥ 0 }`

- 连续时间单位冲激函数`δ(t)`（狄拉克函数）：         

  定义：`δ(t) = 0`（`t ≠ 0`），且满足积分特性 `∫₋∞^∞ δ(t)dt = 1`（积分结果为冲激强度，标准冲激强度为 1）,在0处值趋于无穷，但围成的面积是1        

  筛选特性（核心）：`∫₋∞^∞ x(t) δ(t - t₀)dt = x(t₀)`（提取信号在 `t = t₀` 处的取值）

- 与阶跃函数的关系：连续时间中，单位阶跃函数的导数为单位冲激函数，即 `δ(t) = du(t)/dt`；反之，单位阶跃函数是单位冲激函数的积分，即 `u(t) = ∫₋∞^t δ(τ)dτ`

- 离散时间单位冲激序列 `δ[n]`（单位样值序列）：

  定义：`δ[n] = { 1, n = 0; 0, n ≠ 0 }` 筛选特性：`Σₙ=-∞^∞ x[n] δ[n - n₀] = x[n₀]` 与阶跃序列的关系：`u[n] = Σₖ=-∞^n δ[k]`，`δ[n] = u[n] - u[n - 1]`

### 3. 应用价值

可构成绝大多数实用信号（例如，矩形脉冲可表示为 `x(t) = u(t - t₁) - u(t - t₂)`），是信号表示与系统分析的核心工具，后续LTI系统的冲激响应分析均基于此。

## 六、1.5 连续时间与离散时间系统

### 1. 系统分类

- **连续时间系统**：

  输入 `x(t)`、输出 `y(t)` 均为连续时间信号，其数学模型通常为**线性常系数微分方程**

  例如一阶系统：`a₁ dy(t)/dt + a₀ y(t) = b₀ x(t)`；

  二阶系统：`a₂ d²y(t)/dt² + a₁ dy(t)/dt + a₀ y(t) = b₁ dx(t)/dt + b₀ x(t)`（`a₀,a₁,a₂,b₀,b₁` 为常数）

- **离散时间系统**：

  输入 `x[n]`、输出 `y[n]` 均为离散时间信号，其数学模型通常为线性常系数差分方程

  例如一阶系统：`a₁ y[n] + a₀ y[n - 1] = b₀ x[n]`；

  二阶系统：`a₂ y[n] + a₁ y[n - 1] + a₀ y[n - 2] = b₁ x[n] + b₀ x[n - 1]`

### 2. 系统互联方式

1. **级联**：两个系统 `T₁` 和 `T₂` 级联，前级输出作为后级输入，总系统算子为 `T = T₂{T₁{x(t)}}`，即 `y(t) = T₂{T₁{x(t)}}`，满足交换律（`T₁{T₂{x(t)}} = T₂{T₁{x(t)}}`）
2. **并联**：两个系统 `T₁` 和 `T₂` 接收同一输入，输出相加作为总输出，总系统算子为 `T = T₁ + T₂`，即 `y(t) = T₁{x(t)} + T₂{x(t)}`
3. **反馈联结**：系统输出的一部分通过反馈通路回授至输入端，与原始输入叠加后作为系统输入，典型结构为：输入 `x(t)` 与反馈信号 `f(t) = T₂{y(t)}` 叠加（通常为减法），得到 `e(t) = x(t) - f(t)`，再输入至前向通路系统 `T₁`，输出 `y(t) = T₁{e(t)}`（如智能小车的图像反馈控制）

- 复杂系统由级联、并联、反馈三种方式混合实现，通过分解为简单子系统，可简化复杂系统的分析与设计。

## 七、1.6 系统的基本性质

### 1. 记忆性

- 无记忆系统：任意时刻 `t₀`（或 `n₀`）的输出仅与当前时刻的输入 `x(t₀)`（或 `x[n₀]`）有关，例如 `y(t) = 3x(t)`、`y[n] = x[n] + 2`
- 记忆系统：输出与过去或未来的输入相关，例如 `y(t) = x(t - 1)`（与过去输入相关）、`y[n] = x[n + 2]`（与未来输入相关）、`y(t) = ∫₋∞^t x(τ)dτ`

### 2. 可逆性

- 可逆系统：对任意不同的输入 `x₁(t) ≠ x₂(t)`，必有对应的输出 `y₁(t) ≠ y₂(t)`（输入与输出一一对应），例如 `y(t) = 2x(t)`（可逆）、`y(t) = x²(t)`（不可逆，因 `x(t) = 2` 和 `x(t) = -2` 输出相同）
- 逆系统：若系统 `T` 与系统 `T⁻¹` 级联后构成恒等系统（即 `T⁻¹{T{x(t)}} = x(t)`），则 `T⁻¹` 是 `T` 的逆系统；可逆系统必有唯一逆系统。

### 3. 因果性

- 因果系统：任意时刻 `t₀`（或 `n₀`）的输出仅与当前及此前的输入（`t ≤ t₀` 或 `n ≤ n₀`）有关，不依赖未来输入，所有实时物理系统均为因果系统，例如 `y(t) = x(t) + x(t - 2)`、`y[n] = Σₖ=-∞^n x[k]`
- 非因果系统：输出依赖未来输入，例如 `y(t) = x(t + 1)`（依赖 `t+1` 时刻的输入）、`y[n] = x[n + 3]`；非实时信号处理系统（如离线图像处理）可实现非因果系统。

### 4. 稳定性

- 稳定系统（BIBO 稳定，有界输入有界输出）：若输入`x(t)`（或 `x[n]`）有界（存在常数`M`，使得 `|x(t)| ≤ M` 对所有 `t` 成立，或 `|x[n]| ≤ M` 对所有 `n` 成立），则输出 `y(t)`（或 `y[n]`）也有界，例如 `y(t) = 0.5x(t)`（稳定）
- 不稳定系统：有界输入会产生无界输出，例如 `y(t) = t x(t)`（输入 `x(t) = 1` 有界，输出 `y(t) = t` 无界）；工程中优先设计稳定系统。

### 5. 时不变性

若输入信号时移 `τ`（连续）或`k`（离散），输出信号也产生相同时移，无其他畸变，即：       连续时间：若 `x(t) → y(t)`，则`x(t - τ) → y(t - τ)` 离散时间：若 `x[n] → y[n]`，则 `x[n - k] → y[n - k]` 例如 `y(t) = sin(x(t)`（时不变）、`y(t) = t x(t)`（时变）

### 6. 线性

- 线性系统需同时满足**可加性**和**齐次性**：         

  1. 可加性：若 `x₁(t) → y₁(t)`、`x₂(t) → y₂(t)`，则 `x₁(t) + x₂(t) → y₁(t) + y₂(t)` 

  2. 齐次性（标量乘法）：若 `x(t) → y(t)`，则 `a x(t) → a y(t)`（`a` 为任意常数）         

     合并为线性性质：`a x₁(t) + b x₂(t) → a y₁(t) + b y₂(t)`（`a,b` 为任意常数）

- 线性系统的零输入零输出特性：若输入 `x(t) = 0`（或 `x[n] = 0`），则输出 `y(t) = 0`（或 `y[n] = 0`）；不满足该特性的系统一定是非线性系统。

- **增量线性系统**：输入与输出的增量满足线性关系，即若输入增量为 `Δx(t) = x₂(t) - x₁(t)`，输出增量为 `Δy(t) = y₂(t) - y₁(t)`，则 `a Δx(t) + b Δx'(t) → a Δy(t) + b Δy'(t)`；

  增量线性系统可等效为“线性系统 + 与输入无关的固定响应”，即 `y(t) = T{x(t)} + y₀(t)`（`y₀(t)` 为固定响应，与输入无关），例如 `y(t) = 2x(t) + 3`（增量线性，线性部分为 `2x(t)`，固定响应为 3）。
